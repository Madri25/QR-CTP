# QR CTP - Sistema de Gestión de Transporte Escolar

## 📌 Descripción del proyecto

QR CTP es un sistema desarrollado para resolver el desorden en el proceso de transporte escolar en el colegio CTP La Carpio.

El objetivo principal es **automatizar y optimizar la gestión del transporte utilizando tecnología QR**, mejorando la organización, seguridad y eficiencia en el ingreso y salida de estudiantes.

---

## 🚍 Problema que resuelve

En el entorno escolar existía desorganización en el control de los autobuses, lo que generaba:

- Falta de control de estudiantes en transporte
- Procesos manuales lentos
- Errores en registros
- Baja eficiencia en la supervisión

---

## 💡 Solución

El sistema QR CTP implementa:

- Identificación mediante códigos QR
- Registro digital de usuarios
- Control de acceso al transporte
- Sistema de roles (Administrador, Docente, Estudiante)
- Gestión centralizada de información escolar

---

## 👥 Roles del sistema

### 🛠️ Administrador
- Gestión completa del sistema
- Administración de usuarios
- Control de registros
- Acceso total a módulos

### 👨‍🏫 Docente
- Supervisión de estudiantes
- Validación de accesos
- Consulta de información

### 🎓 Estudiante
- Acceso mediante QR
- Visualización de información personal
- Uso del sistema de transporte

---

## 🧰 Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- PHP
- MySQL
- XAMPP (desarrollo inicial)
- Laragon (entorno posterior)

---

## 🖥️ Funcionalidades principales

- Sistema de login con validación de usuarios
- Escaneo de QR para control de transporte
- Paneles según rol del usuario
- Interfaz web adaptable a múltiples dispositivos (responsive design)
- Conexión a base de datos para almacenamiento de información
